## 下载

### Docker

```
docker pull apachecn0/programiz-zh
docker run -tid -p <port>:80 apachecn0/programiz-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g programiz-zh
programiz-zh <port>
# 访问 http://localhost:{port} 查看文档
```
