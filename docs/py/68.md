## 源代码

```py
# This program prints Hello, world!

print('Hello, world!') 
```

**输出**

```py
Hello, world!

```

在此程序中，我们使用了内置的`print()`函数在屏幕上打印字符串`Hello, world!`。

顺便说一下，[字符串](/python-programming/string "Python Strings")是一个字符序列。 在 Python 中，字符串用单引号，双引号或三引号括起来。
