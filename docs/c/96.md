## 使用递归的两个数的 GCD

```c
#include <stdio.h>
int hcf(int n1, int n2);
int main() {
    int n1, n2;
    printf("Enter two positive integers: ");
    scanf("%d %d", &n1, &n2);
    printf("GCD of %d and %d is %d.", n1, n2, hcf(n1, n2));
    return 0;
}

int hcf(int n1, int n2) {
    if (n2 != 0)
        return hcf(n2, n1 % n2);
    else
        return n1;
} 
```

**输出**

```c
Enter two positive integers: 366
60
GCD of 366 and 60 is 6. 
```

在此程序中，进行递归调用，直到`n2`的值等于 0。
