## 不使用 strlen（）函数即可计算字符串的长度

```c
#include <stdio.h>
int main() {
    char s[] = "Programming is fun";
    int i;

    for (i = 0; s[i] != '\0'; ++i);

    printf("Length of the string: %d", i);
    return 0;
} 
```

**输出**

```c
Length of the string: 18 
```

在这里，我们使用`for`循环从`i = 0`到`'\0'`（空字符）循环遍历字符串。 在每次迭代中， `i` 的值增加 1。

循环结束时，字符串的长度将存储在 `i` 变量中。
